> 本插件基于 WebGPU 技术与离线 AI 模型实现。为了保证推理效率，运行环境需满足一定的硬件性能要求。

> sszm_whisper 方案，采用轻量化 Whisper 模型，兼顾速度与资源占用。
> 
> sszm_voxtral.realtime 方案，采用高性能 Voxtral 模型，识别效果出众，接近 Chrome 原生实时字幕水平。
> 
> 高性能模型（Voxtral）对硬件要求较高，经实测，当前版本的显存占用最低约为 5GB。
> 
> 整合 Chrome 内置 AI 引擎，支持外语到中文的即时翻译。
 
> 首次切换语言将触发后台自动下载翻译模型。若弹出层（Popup）底部长时间处于“启动中...请稍等”状态，请尝试多次点击“开启字幕”按钮以唤醒初始化流程。
